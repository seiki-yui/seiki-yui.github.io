关于

Seiki Bot 是正在开发使用的私人娱乐聊天bot. 使用 Mirai 开发.

Seiki 来自系列同人游戏 东方Project 中游戏 东方辉针城 二面角色 赤蛮奇 日文名罗马音 “Seki” 的错译.

使用

目前 Seiki Bot 已在 Github 开源. 下面是其使用指南.
通过发送相对应指令来触发bot.

带括号的/前缀是可选的
<...>中的是指令名，由空格隔开表示或，选择其中任一名称都可执行例如#yinglish,#yl
[...]表示参数，当[...]后面带?时表示参数可选,当其为[用户]时支持使用@,QQ号及昵称匹配.
{...}表示连续的多个参数

PS:

  • 使用指令请在每条指令前加上井号”#”.
  • 参数为字符时,如有空格需要请加上双引号”。
  • [...]为[用户]时,还支持~作为发送者自己,$作为随机群员.

例:

  • 如果你想生成肯德基疯狂星期四文案, 发送#kfc即可;
  • 如果你想让答案之书回答你的问题, 发送#dazs 问题,把问题改成你自己的,例如#dazs 我做东方鬼畜音mad好吗?;
  • 如果你想生成五千兆円图片, 发送#5k 文字1 文字2.如果文字带空格, 请使用双引号包裹, 如#5k "Hello World!" "V几百 你你你你 啾啾啾"

Plain 文本类

指令 描述
#<ping> [网址]? [次数]? Ping[网址],为空时默认为Bot的服务器.
#<gpbt 狗屁不通> [文本] [长度]? 生成狗屁不通文章.
#<form 组成> [名字] 看看你的成分构成!
#<2cy 二次元> [名字] 二次元少女的你!
#<dazs 答案之书> [问题] 让答案之书回答你的问题.
#<kfc 肯德基> 生成肯德基疯狂星期四文案
#<diana 发病> [名字] 发病小作文,[名字]默认为然然.
#<yiyan 1y 一言> 手动获取一条一言.
#<nbnhhsh hhsh 能不能好好说话> [缩写] 查询网络用语[缩写](仅英文)的含义.
#<baike 百科> [内容] 百度百科.
#<bottle> <get> 捡一条漂流瓶.
#<bottle> <throw> [标题] [内容] 扔一条漂流瓶.
(#)<fencing 击剑> [用户] 击剑:@一个人，随机禁言其中一个
#<yinglish yl 淫语> [文本] 淫…淫语翻…翻译器…♡
